admbook

features

  • база данных не требуется
  • позволяет удалять и редактировать сообщения через браузер (admin only)
  • позволяет админу отвечать на сообщения через браузер (admin only)
  • не позволяет использовать html тэги, кроме: <a href="http://domain.com">Domain</a>
    <b>some text</b> <i>some text</i> (включить/отлючить можно через браузер (admin only))
  • разделяет слова на части по nn символов (количество символов в сообщении можно изменить через браузер (admin only))
  • разбивает сообщения на страницы (количество сообщений на странице можно изменить через браузер (admin only))
  • заменяет смайлики на картинки (например ;) - или :p - ) (включить/отлючить можно через браузер (admin only))
  • есть фильтр нежелательных слов из словаря (dictionary.inc - слова можно добавлять в словарь через браузер (admin only))

install

  • скопировать все файлы в одну дерикторию, включая поддериктории
  • у всех файлов, кроме data.inc, dictionary.inc и data.php выставить права, командой chmod 755 *, подробнее о выставлении прав на файл можно почитать на php.sbp.ru или citforum
  • у файла data.inc выставить права, командой chmod 666 data.inc
  • у файла dictionary.inc выставить права, командой chmod 666 dictionary.inc
  • у файла data.php выставить права, командой chmod 666 data.php
  • dictionary.inc - файл содержит слова, которые нежелательно отображать, одна строка, считается одним словом.
  • в разделе configuration админской части выставить charset : win-1251
  • в разделе configuration админской части выставить locale : RU
