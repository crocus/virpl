Работа функции rdate подобна функции date в РНР,
за исключением того что вывод названия недели и месяца,
на Русском языке.


Форматы для использования в функции rdate :

---------------------------------------------------------

	w	// Назване недели на русском языке
	m	// Название месяца на русском языке
	H	// Час (формат 1 - H)
	i	// Минуты (формат 1 - i)
	Y	// Год - полный формат
	y	// Год - короткий формат
	d	// Число
	D	// Сокращенное обозначение дня недели
	M	// Сокращенное обозначение месяца
	s	// секунды

--------------------------------------------------------

При этом позволяется использования обычного текста в шаблоне:
Например, ввести в качестве параметра rdate следующию строку:
echo RDate("Сегодня w d m Y года");

То на выводе получим такое:
Сегодня воскресенье 30 марта 2011 года
